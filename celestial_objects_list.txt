## NOTES FOR ARTWORK GENERATION

- Maintain consistent style across all objects
- Use INSPIRE brand colors: electric purple (#a855f7), neon teal (#06b6d4), neon magenta (#ec4899)
- Keep backgrounds dark cosmic black
- Add subtle glows and halos
- Make each object instantly recognizable
- Include some mystical/spiritual energy
- Sci-fi guerrilla aesthetic throughout
- Bold, clean lines
- Each image should work at 200x200px minimum
- Consider adding small constellation patterns or reference markers
- Emoji can supplement but shouldn't replace artwork
- File naming: [object-name]-artwork.png (e.g., "sirius-artwork.png")
